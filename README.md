# UniFlow AI

UniFlow AI es una plataforma web orientada a estudiantes que necesitan organizar tareas, proyectos y exámenes de forma más inteligente. El sistema utiliza una lógica básica inspirada en Inteligencia Artificial para analizar la dificultad y urgencia de una actividad, generar una prioridad y recomendar qué acción tomar.

## Descripción del proyecto

Muchos estudiantes suelen tener varias actividades pendientes al mismo tiempo, como tareas, exámenes, proyectos, exposiciones o entregas importantes. Sin una buena organización, es común dejar trabajos para el último momento, olvidar fechas límite o no saber qué actividad atender primero.

UniFlow AI busca resolver este problema mediante una página web sencilla donde el estudiante puede registrar una tarea y recibir una recomendación automática basada en su nivel de dificultad y urgencia.

## Objetivo

Desarrollar una plataforma web con Inteligencia Artificial que ayude a estudiantes a organizar sus actividades académicas, priorizar tareas y reducir el riesgo de atraso en entregas importantes.

## Problema que resuelve

El proyecto atiende una problemática común en estudiantes: la mala organización del tiempo académico. Cuando existen múltiples tareas y fechas de entrega, puede ser difícil decidir qué hacer primero. UniFlow AI propone una solución digital que analiza los pendientes y genera recomendaciones para mejorar la planificación.

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/uniflow-ai.git
```

2. Entrar a la carpeta del proyecto:

```bash
cd uniflow-ai
```

3. Abrir el archivo `index.html` en el navegador.

También se puede ejecutar usando la extensión Live Server de Visual Studio Code.

## Uso básico

1. Abrir la página principal.
2. Ir a la sección de demo.
3. Escribir el nombre de una tarea.
4. Seleccionar la dificultad.
5. Seleccionar la urgencia.
6. Presionar el botón "Analizar tarea".
7. Revisar la prioridad y recomendación generada por el sistema.

## Ejemplo de uso

Entrada del usuario:

- Tarea: Proyecto de programación
- Dificultad: Alta
- Urgencia: Alta

Resultado esperado:

- Clasificación: Prioridad máxima
- Recomendación: trabajar en esta tarea lo antes posible, dividirla en partes pequeñas y dedicarle tiempo durante el día.

## Propuesta de Inteligencia Artificial

En esta primera versión, el sistema utiliza una lógica básica de reglas para simular el análisis inteligente de tareas. En una versión futura, se podría integrar un modelo de Inteligencia Artificial más avanzado capaz de:

- Predecir riesgo de atraso.
- Generar calendarios automáticos.
- Recomendar horarios de estudio.
- Clasificar tareas según importancia.
- Analizar hábitos del estudiante.
- Crear planes semanales personalizados.

## Público objetivo

UniFlow AI está dirigido principalmente a:

- Estudiantes universitarios.
- Estudiantes de preparatoria.
- Personas con múltiples entregas académicas.
- Usuarios que desean mejorar su organización personal.

## Valor del proyecto

El valor principal de UniFlow AI está en ayudar a los estudiantes a tomar mejores decisiones sobre el uso de su tiempo. A diferencia de una agenda tradicional, el sistema no solo guarda tareas, sino que también analiza su urgencia y dificultad para sugerir qué actividad debería atenderse primero.
